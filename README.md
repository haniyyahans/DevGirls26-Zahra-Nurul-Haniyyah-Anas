# StudyTracker

##📖 Deskripsi Aplikasi

StudyTracker adalah aplikasi to-do list sederhana yang dirancang untuk membantu mahasiswa mencatat tugas atau kegiatan belajar harian, menandai progres, dan melihat ringkasan produktivitas mereka secara simpel. Aplikasi ini berbasis frontend-only — tidak memerlukan server atau koneksi internet untuk berjalan. Seluruh data bersumber dari input pengguna dan disimpan secara lokal di perangkat masing-masing menggunakan Local Storage browser, sehingga data tetap tersimpan meskipun halaman di-refresh.

##✨ Fitur Utama

* Dashboard Produktivitas — Menampilkan ringkasan jumlah total tugas, tugas selesai, dan tugas belum selesai secara real-time.
* Progress Bar — Visualisasi persentase penyelesaian tugas hari ini.
* Tambah Tugas — Input nama tugas, deskripsi singkat (opsional), dan deadline.
* Deadline Badge — Indikator warna otomatis: normal, segera (≤ 3 hari), atau lewat deadline.
* Filter Tugas — Tampilkan semua tugas, hanya yang belum selesai, atau yang sudah selesai.
* Tandai Selesai — Klik checkbox untuk menandai tugas sebagai selesai atau membatalkannya.
* Hapus Tugas — Hapus tugas yang tidak diperlukan dengan konfirmasi.
* Persistent Storage — Data tersimpan otomatis di Local Storage browser.
* Responsif — Tampilan menyesuaikan layar desktop maupun mobile.

##🛠️ Teknologi yang Digunakan

* HTML5 — Struktur halaman dan elemen antarmuka
* CSS3 — Styling, animasi, dan desain responsif dengan CSS Variables
* Javascript — Logika aplikasi, manipulasi DOM, dan pengelolaan state
* Local Storage API — Penyimpanan data tugas secara persisten di browser
* Google Fonts — Tipografi
* Claude AI — Membantu menyempurnakan dan memberi saran

##🚀 Cara Menjalankan Aplikasi

Karena StudyTracker adalah aplikasi berbasis frontend-only, maka cara menjalankannya:
1. Unduh atau simpan file study-tracker.html ke perangkat kamu.
2. Buka file tersebut menggunakan browser (Chrome, Firefox, Edge, Safari, dll.).
3. Aplikasi langsung siap digunakan — tidak perlu instalasi apapun.
